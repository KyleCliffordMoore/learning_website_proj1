Nutrinotes

This project's goal was to create a website where you could create a "food catelog" in which you could put all your favorite foods into it or have a catelog for "junk food", "healthy food", etc. Then Some other user could browse catelogs and decide if they like your catelog and copy it or add foods to make their own. The site also had a friends system so that you could friend another user and the site would show you how long the both of you had been friends.

This project involed mostly html with a small amount of javascript for the front end. The backend was made with Python using Flask and GUnicorn to run the webpage which was hosted on render.com. The backend also included Postgres SQL to save and retrieve data. The github repository can be found here:

https://github.com/Summer24-CSCE-310-Database-Systems/mvp-nutrinotes

3D Renderer in Java

This project's goal was to create a simple 3D renderer in Java usiing only Java Swing and no external libraries. The point of this project was to better understand the math behind 3d renderers so I decided to see how far I could go in making my own. This project is still a work in progress as I am still researching about backface culling methods to try and fix the problem where some points are out of view of the camera and others are not.

The github repository can be found here: https://github.com/KyleCliffordMoore/JavaBasic3DRenderer

Photo to Desmos

This project's goal was to have some fun by making something that could take an image and return a set of Bézier curves that would form an image that could be rendered in Desmos.

This project required a lot of research into how Bézier curves worked as well as how Gaussian blurs could help in edge detection. For this project I used two external libraries: OpenCV which was used to apply blurs, gray scaling, and edge detection. Potrace which was used to take bitmap images and convert them into a vector image file (I used .svg). I then made a basic .svg interpreter to build the Bézier curves from the file data.

The github repository can be found here: https://github.com/KyleCliffordMoore/Desmos-animation
